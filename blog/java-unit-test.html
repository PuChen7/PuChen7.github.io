Unit Test With JUnit

Introduction

Unit testing involves breaking your program into pieces, and subjecting each piece to a series of tests.

JUnit
Offical Website
JUnit is a unit testing framework for the Java programming language.
To use JUnit, import following packages:
import org.junit.Test;
import static org.junit.Assert.*;
Sample Test program:

                    import org.junit.Test;
                    import static org.junit.Assert.*;
                    public class MyUnitTest {
                        @Test
                        public void testConcatenate() {
                            MyUnit myUnit = new MyUnit();
                            String result = myUnit.concatenate("one", "two");
                            assertEquals("onetwo", result);
                        }
                    }
The method annotated with @Test will be executed by unit test runner.

Assert Methods
List of methods:

  • assertArrayEquals()
  • assertEquals()
  • assertTrue() + assertFalse()
  • assertNull() + assertNotNull()
  • assertSame() + assertNotSame()
  • assertThat()
assertArrayEquals()
The assertArrayEquals() method will test whether two arrays are equal to each other. In other words, if the two arrays contain the same number of elements, and if all the elements in the array are equal to each other.
The elements of each array are compared one by one using their equals() method.
That means, that it is not enough that the two arrays contain the same elements. They must also be present in the same order.
